# web-crawler
web site crawler through qt5 designer

### 실행방법
python main.py

원하는 웹페이지 버튼 클릭 후 OK > 크롤러 csv파일 생성

## purpose
http://www.llumar.co.kr/

## result
What's new, FAQ, Promotion page webcrawler csv file.
